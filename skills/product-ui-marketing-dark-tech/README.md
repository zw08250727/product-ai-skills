# product-ui-marketing-dark-tech

Desktop-first product UI and dark-tech visual guidance for SaaS, AI products, dashboards, consoles, and high-fidelity prototypes.

## What It Helps With

- making product pages feel more credible and commercial
- avoiding generic landing-page layouts
- improving desktop information density without losing clarity
- designing workflow-heavy and dashboard-heavy interfaces
- refining dark UI systems with better hierarchy and restraint

## Best-Fit Scenarios

- SaaS marketing pages
- AI product homepages
- B2B admin consoles
- product dashboards
- onboarding and login surfaces
- single-file HTML prototypes

## Trigger Phrases

Typical requests that fit this skill:

- make this feel more polished
- redesign this as a real SaaS product page
- make it more desktop-first
- give it a dark-tech product feel
- improve this dashboard or workflow UI

## Output Style

The skill is optimized to push AI toward:

- clearer product storytelling
- stronger PC layout composition
- more realistic product screenshots and dashboards
- restrained but useful visual effects
- reusable visual tokens instead of scattered one-off styles

## Entry File

- `SKILL.md`

## Notes

This skill intentionally prioritizes product clarity over decoration.
