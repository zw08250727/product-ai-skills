# Product AI Skills

Reusable AI skills for product managers, AI PMs, prototyping workflows, and product-led interface design.

This repository is a local open-source preparation workspace. The first extracted skills are:

- `product-ai-prototyping`
- `product-ui-marketing-dark-tech`

It is designed for:

- PC-first SaaS landing pages
- AI product pages
- B2B dashboards and consoles
- high-fidelity HTML prototypes
- workflow-heavy product interfaces

## Included Skill

### `product-ai-prototyping`

Focus:

- turning fuzzy product ideas into structured requirements
- scoping PRDs and solution directions
- separating scenarios, roles, objects, and workflows
- preparing prototype-ready briefs
- generating reusable AI prompt templates for PM work

Use it when you want AI to help with product discovery, feature framing, workflow design, PRD scoping, prototype planning, or system-boundary thinking before jumping into UI.

### `product-ui-marketing-dark-tech`

Focus:

- product UI refinement
- dark-tech visual direction
- desktop-first layout strategy
- more credible product screens
- landing pages with actual product narrative

Use it when you want AI to improve a page, prototype, dashboard, or SaaS interface so it feels more productized, more polished, and more suitable for desktop presentation.

## Installation Notes

Different AI tools load skills differently, but the common pattern is:

1. Copy the skill folder into your local skills directory.
2. Keep the standard `SKILL.md` entrypoint.
3. Add supporting references next to it when needed.

Example target layout:

```text
<your-skills-root>/
└── product-ui-marketing-dark-tech/
    └── SKILL.md
```
