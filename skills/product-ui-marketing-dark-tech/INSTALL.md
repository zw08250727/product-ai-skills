# 安装方式

本 skill 适用于 WorkBuddy / Claude / Codex 等 AI 辅助工具。

## 安装步骤

1. 将本目录复制到你的 skills 目录：
   - WorkBuddy: `~/.workbuddy/skills/`
   - Claude: `~/.claude/skills/`
   - Codex: 对应 skills 目录

2. 重新加载 AI 工具即可使用

## 文件说明

- `SKILL.md` — skill 主定义文件
- `README.md` — 功能说明
- `references/` — 相关参考资料（可选）

## 注意事项

- 本 skill 为纯描述型，无需额外安装脚本或依赖
- 请确保 AI 工具支持对应格式的 skill
