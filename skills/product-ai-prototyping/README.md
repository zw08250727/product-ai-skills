# product-ai-prototyping

Turn vague product ideas into structured requirements, prototype plans, and reusable AI prompts for product work.

## What It Helps With

- product discovery
- feature scoping
- PRD framing
- scenario and role design
- object and workflow modeling
- frontend vs backend boundary decisions
- prototype-ready solution planning

## Best-Fit Scenarios

- you have a rough product idea but not a clean structure yet
- you want AI to help shape a feature before drawing screens
- you need a more rigorous product plan instead of a loose brainstorm
- you want reusable prompt patterns for PM work

## Output Style

This skill pushes AI toward a more structured PM workflow:

- lock the parent product first
- define goals and non-goals
- split scenarios before pages
- split roles before actions
- model objects and states before UI
- reason about system boundaries before polishing screens
